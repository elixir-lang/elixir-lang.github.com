escape/1

Recursively escapes the given value so it can be inserted into a syntax tree. Structures that are valid syntax nodes (like atoms, integers, binaries) are represented by themselves.

Examples

Macro.escape(:foo)
#=> :foo

Macro.escape({ :a, :b, :c })
#=> { :{}, 0, [:a, :b, :c] }
