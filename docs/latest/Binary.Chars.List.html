to_binary/1

Consider the list is an iolist and converts it to a binary. This allows a list of binaries, or a charlist, or a mix of both, to be converted successfully.

Examples

to_binary 'foo' #=> "foo"
to_binary ["foo", 'bar'] #=> "foobar"
