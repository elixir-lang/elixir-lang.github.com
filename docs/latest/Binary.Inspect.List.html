inspect/1

Represents a list checking if it can be printed or not. If so, a single-quoted representation is returned, otherwise the brackets syntax is used.

Examples

inspect('bar')       #=> 'bar'
inspect([0|'bar'])   #=> "[0,98,97,114]"
inspect([:foo,:bar]) #=> "[:foo, :bar]"
