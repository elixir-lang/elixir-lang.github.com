inspect(tuple, opts)

Represents tuples. If the tuple represents a record, it shows it nicely formatted using the access syntax.

Examples

iex> inspect({1, 2, 3})
"{1, 2, 3}"
iex> inspect(ArgumentError.new)
"ArgumentError[message: \"argument error\"]"
