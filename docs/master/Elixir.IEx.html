Elixir.IEx

This module implements interactive Elixir. It provides two main functions, start and simple_start. start was meant for systems where tty is available and relies on it in order to work properly. This makes all control commands available in tty available to the developer.

In case tty is not available (for example, Windows), a developer may invoke simple_start which starts a stripped down version.
